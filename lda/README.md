# [GibbsLDA++](http://gibbslda.sourceforge.net/)

**GibbsLDA++** is a C/C++ implementation of Latent Dirichlet Allocation (LDA) using Gibbs Sampling technique for parameter estimation and inference.

## Compile

```bash
make clean
make all
```
